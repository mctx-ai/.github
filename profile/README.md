## About mctx

mctx is a hosted MCP server platform with built-in developer monetization. Connect your GitHub repo containing an MCP server, and we handle hosting, authentication, and payments.

### The Three Pillars

- **Free MCP Hosting** — Deploy your MCP servers at no cost. We handle infrastructure, scaling, and uptime.
- **Distribution** — One-click publish to the MCP Community Registry and GitHub registries. Reach developers building with Claude and other AI tools.
- **Set Your Price. Get Paid.** — Earn revenue from subscriptions. You keep 80%, we take 20% and handle the rest.

## Getting Started

1. **Visit** https://mctx.ai
2. **Sign up** with your GitHub account
3. **Set up payouts** via Stripe Connect
4. **Connect your repository** — Link your GitHub repo with an MCP server
5. **Deploy** — mctx builds and hosts your server automatically
6. **Earn** — Subscribers pay, you receive payouts automatically
